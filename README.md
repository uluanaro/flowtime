# FlowTime — AI-powered Task Scheduler

## Описание
SaaS приложение для управления задачами с AI планировщиком.

## Стек
- **Backend:** Python, FastAPI, SQLite
- **Frontend:** React, Vite
- **AI:** Groq API (LLaMA 3.3)

## Функциональность
- CRUD задач
- Управление расписанием
- AI оценка длительности задач
- AI планирование дня по свободным слотам
- Разбивка больших целей на подзадачи
